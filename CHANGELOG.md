## [1.0.0] — 2025-06-01

### Added

**Core Framework Documentation**
- `README.md` — Project overview, structure, and quick start guide
- `GOVERNANCE.md` — Agora Framework governance model, decision-making processes, and stewardship roles
- `CODE_OF_CONDUCT.md` — Community standards based on the Contributor Covenant v2.1, adapted for institutional contexts
- `CONTRIBUTING.md` — Comprehensive contribution guidelines covering documentation, policy templates, and process improvements
- `CHANGELOG.md` — This document
- `publiccode.yml` — Machine-readable metadata for public administration software catalogues (compliant with publiccode.yml standard v0.3)
- `.github/PULL_REQUEST_TEMPLATE.md` — Standardised pull request submission template

**White Paper**
- `docs/whitepaper/agora-whitepaper.md` — Full white paper: *Agora: A Reference Framework for Open Source Program Offices in Public and Mission-Driven Institutions*
- `docs/whitepaper/agora-whitepaper.docx` — Word/OOXML version for institutional distribution
- `docs/whitepaper/agora-whitepaper.odt` — ODF version for open-source office suite compatibility

**Policy Templates**
- `docs/templates/ospo-policy-template.md` — Institutional OSPO Policy Template (Markdown)
- `docs/templates/ospo-policy-template.docx` — Institutional OSPO Policy Template (Word/OOXML)
- `docs/templates/ospo-policy-template.odt` — Institutional OSPO Policy Template (ODF)

**Communication Materials**
- `articles/linkedin-article-1.md` — LinkedIn article: *Introducing the Agora Framework*
- `articles/linkedin-article-2.md` — LinkedIn article: *The OSPO as Institutional Infrastructure*

### Framework Coverage (v1.0.0)
- Institutional mandate definition
- OSPO policy architecture (6 policy domains)
- Licence compliance workflow aligned with OpenChain ISO/IEC 5230
- Contribution governance model (inbound / outbound / inner source)
- Community engagement framework
- CHAOSS-aligned metrics and KPI catalogue
- European regulatory alignment matrix (EIF, NIS2, GDPR, EU OSS Strategy)
